Räkneövning 1 - Uppgift hö1 från [1]

Två linjärt termo-elastiska massiva cylindrar (Elasticitetsmodul EEE och linjär temperaturutvidgningskoefficient α\alphaα) med mått enligt figur är med exakt passning monterade mellan två stela väggar. Ställ upp de ekvationer som behövs för att bestämma reaktionskrafter i väggarna om den större cylindern värms T∘T^\circT∘C.

Lösning

Vi börjar med att frilägga och införa snittkrafter enligt figur.

Kraftjämvikt ger: →: −N1+N2=0⇒−σ1A1+σ2A2=0. \rightarrow:\ -N_1 + N_2 = 0 \qquad \Rightarrow \qquad -\sigma_1 A_1 + \sigma_2 A_2 = 0. →: −N1​+N2​=0⇒−σ1​A1​+σ2​A2​=0.

För att gå vidare ställer vi upp materialsamband. För linjärt elastiskt material får vi ε1=σ1E+αΔT1=σ1E, \varepsilon_1 = \frac{\sigma_1}{E} + \alpha \Delta T_1 = \frac{\sigma_1}{E}, ε1​=Eσ1​​+αΔT1​=Eσ1​​, ε2=σ2E+αΔT2=σ2E+αT, \varepsilon_2 = \frac{\sigma_2}{E} + \alpha \Delta T_2 = \frac{\sigma_2}{E} + \alpha T, ε2​=Eσ2​​+αΔT2​=Eσ2​​+αT, där ε1\varepsilon_1ε1​ och ε2\varepsilon_2ε2​ är den totala töjningen i respektive del. Töjningen kan skrivas som kvoten mellan förlängning och ursprungslängd. Vi definierar δ1\delta_1δ1​ och δ2\delta_2δ2​ till att vara de båda delarnas förlängning. Eftersom stångsystemet är fast mellan väggarna måste den totala förlängningen vara 0, dvs δ1+δ2=0⇒δ=δ1=−δ2. \delta_1 + \delta_2 = 0 \qquad \Rightarrow \qquad \delta = \delta_1 = -\delta_2. δ1​+δ2​=0⇒δ=δ1​=−δ2​. Vi kan nu skriva töjningen i varje del som ε1=δ1L1=δ2L, \varepsilon_1 = \frac{\delta_1}{L_1} = \frac{\delta}{2L}, ε1​=L1​δ1​​=2Lδ​, ε2=δ2L2=−δ3L. \varepsilon_2 = \frac{\delta_2}{L_2} = -\frac{\delta}{3L}. ε2​=L2​δ2​​=−3Lδ​.

Vi har nu 6 ekvationer och 6 obekanta (N1N_1N1​, N2N_2N2​, σ1\sigma_1σ1​, σ2\sigma_2σ2​, δ1\delta_1δ1​, δ2\delta_2δ2​) ⇒\Rightarrow⇒ lösbart system!
