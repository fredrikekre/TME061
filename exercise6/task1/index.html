Räkneövning 6 - Uppgift 6.34 från [1]

Lös elastiska linjens ekvation för problemet som beskrivs i figuren. Använd resultatet för att beräkna stödreaktionerna RAR_ARA​, RBR_BRB​ och MAM_AMA​. Lastintensiteten är q(x)=−Q/Lq(x) = -Q / Lq(x)=−Q/L.

Lösning

Elastiska linjens ekvation ges av följande 4e ordnings diff. ekv. d2dx2(E(x)I(x)d2w(x)dx2)=q(x) \frac{\mathrm{d}^2}{\mathrm{d}x^2} \left(E(x)I(x) \frac{\mathrm{d}^2 w(x)}{\mathrm{d}x^2}\right) = q(x) dx2d2​(E(x)I(x)dx2d2w(x)​)=q(x)

Vanligtvis beror inte EEE och III av xxx, och ekvation (1) kan då förenklas till

d4w(x)dx4=1EIq(x). \frac{\mathrm{d}^4 w(x)}{\mathrm{d}x^4} = \frac{1}{EI} q(x). dx4d4w(x)​=EI1​q(x). I detta fallet är lasten dessutom konstant d4w(x)dx4=1EI−QL=−QEIL \frac{\mathrm{d}^4 w(x)}{\mathrm{d}x^4} = \frac{1}{EI}\frac{-Q}{L} = \frac{-Q}{EIL} dx4d4w(x)​=EI1​L−Q​=EIL−Q​ En allmän lösning fås genom att integrera ekvation (3) 4 gånger för att bli av med alla derivator, vilket ger

w(x)=−QEILx424+C1x36+C2x22+C3x+C4. w(x) = \frac{-Q}{EIL}\frac{x^4}{24} + C_1 \frac{x^3}{6} + C_2 \frac{x^2}{2} + C_3 x + C_4. w(x)=EIL−Q​24x4​+C1​6x3​+C2​2x2​+C3​x+C4​.

Nu återstår endast att bestämma integrationskonstanterna C1−C4C_1 - C_4C1​−C4​, vilket görs med hjälp av randvillkor. Randvillkoren utläses ur uppgiften. Till vänster är balken fast inspänd vilket innebär att både utböjning www och derivata dwdx\frac{\mathrm{d}w}{\mathrm{d}x}dxdw​ är 000. Till höger är balken fritt upplagd, vilket innebär att utböjning www och moment MMM är 000. Detta ger följande ekvationer för att lösa ut integrationskonstanterna:

w(x=0)=0, w(x = 0) = 0, w(x=0)=0, dwdx(x=0)=0, \frac{\mathrm{d}w}{\mathrm{d}x}(x = 0) = 0, dxdw​(x=0)=0, w(x=L)=0, w(x = L) = 0, w(x=L)=0, M(x=L)=−EId2wdx2(x=L)=0. M(x = L) = -EI\frac{\mathrm{d}^2w}{\mathrm{d}x^2}(x = L) = 0. M(x=L)=−EIdx2d2w​(x=L)=0.

Systemet löses med hjälp av MATLAB symbol funktionalitet nedan.

MATLAB kod

% Uppgift 6.34

clear all; close all

% Definiera symboliska variabler
syms C1 C2 C3 C4 w(x) L EIy q(x) Q

% Utbredd last (konstant)
q(x) = -Q/L;

% Allmän lösning till elastiska linjens d.e.
w(x) = C1*x^3/6 + C2*x^2/2 + C3*x + C4 + int(int(int(int(q/EIy, x), x), x), x);
wp(x) = diff(w, x);

% Böjmoment
M(x) = -EIy * diff(diff(w, x), x);

% Tvärkraft
T(x) = -EIy * diff(diff(diff(w, x), x), x);
% Alternativt T(x) = diff(M, x)

% Bestämning av integrationskonstanter
[C1_,C2_,C3_,C4_] = solve(...
                          w(0)  == 0, ...
                          wp(0) == 0, ...
                          w(L)  == 0, ...
                          M(L)  == 0, ...
                          C1, C2, C3, C4);

% Utskrift av lösning på integrationskonstanter
disp(['C1 = ' char(simplify(C1_))])
disp(['C2 = ' char(simplify(C2_))])
disp(['C3 = ' char(simplify(C3_))])
disp(['C4 = ' char(simplify(C4_))])

% Sätt in integrationskonstanter i utböjningen, böjmoment och tvärkraft
w_(x) = simplify(subs(w, [C1, C2, C3, C4], [C1_, C2_, C3_, C4_]))
M_(x) = simplify(subs(M, [C1, C2, C3, C4], [C1_, C2_, C3_, C4_]))
T_(x) = simplify(subs(T, [C1, C2, C3, C4], [C1_, C2_, C3_, C4_]))

% Reaktionskrafter och reaktionsmoment
RA = - T_(0)
RB = T_(L)
MA = M_(0)
