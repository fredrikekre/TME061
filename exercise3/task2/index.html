σ=(120−400−400000−100) MPa \boldsymbol{\sigma} = \begin{pmatrix} 120 && -40 && 0 \\ -40 && 0 && 0 \\ 0 && 0 && -100 \end{pmatrix} \ \mathrm{ MPa} σ=⎝⎜⎛​120−400​​−4000​​00−100​⎠⎟⎞​ MPa

Beräkna (a) huvudspänningar och (b) normal- och skjuvspänning i (x-y)-planet längs ett snitt som lutar 30∘30^\circ30∘ mot x-axeln (se figur).

Lösning

Huvudspänningar kan fås fram genom Mohrs cirkel (se tidigare uppgift) eller genom att lösa egenvärdesproblemet:

(σ−λI)⋅n=0 (\boldsymbol{\sigma} - \lambda \boldsymbol{I}) \cdot \boldsymbol{n} = 0 (σ−λI)⋅n=0 vilket har en icke-trivial lösning då determinanten är 0:

det⁡(σ−λI)=0. \det (\boldsymbol{\sigma} - \lambda \boldsymbol{I}) = 0. det(σ−λI)=0.

Detta expanderas till det⁡(120−λ−400−40−λ000−100−λ)=(−100−λ)det⁡(120−λ−40−40−λ)=(−100−λ)((120−λ)(−λ)−402)=0 \det \begin{pmatrix} 120-\lambda && -40 && 0 \\ -40 && -\lambda && 0 \\ 0 && 0 && -100-\lambda \end{pmatrix} = (-100-\lambda) \det \begin{pmatrix} 120-\lambda && -40\\ -40 && -\lambda \end{pmatrix} = (-100-\lambda) ((120-\lambda)(-\lambda)- 40^2) = 0 det⎝⎜⎛​120−λ−400​​−40−λ0​​00−100−λ​⎠⎟⎞​=(−100−λ)det(120−λ−40​​−40−λ​)=(−100−λ)((120−λ)(−λ)−402)=0

Ur detta ser vi direkt att λ1=−100 MPa\lambda_1 = -100\ \mathrm{ MPa}λ1​=−100 MPa är en rot. De övriga fås ur en andragradsekvation ((120−λ)(−λ)−402)=(λ2−120λ−402)=0⇒λ2=−12 MPa, λ3=132 MPa. ((120-\lambda)(-\lambda)- 40^2) = (\lambda^2 - 120\lambda - 40^2) = 0 \quad \Rightarrow \quad \lambda_2 = -12\ \mathrm{ MPa},\ \lambda_3 = 132\ \mathrm{ MPa}. ((120−λ)(−λ)−402)=(λ2−120λ−402)=0⇒λ2​=−12 MPa, λ3​=132 MPa.

Spänningsvektorn för ett plan fås genom projektion. Vi behöver en normalvektor vinkelrätt mot planet, n⊥\boldsymbol{n}_\perpn⊥​:

n⊥=(−sin⁡30∘cos⁡30∘0) \boldsymbol{n}_\perp = \begin{pmatrix} -\sin 30^\circ\\ \cos 30^\circ\\ 0 \end{pmatrix} n⊥​=⎝⎜⎛​−sin30∘cos30∘0​⎠⎟⎞​

Spänningsvektorn fås då som σ⋅n⊥. \boldsymbol{\sigma} \cdot \boldsymbol{n}_\perp. σ⋅n⊥​.

För att få fram normalspänningen projicerar vi spänningsvektorn på n⊥\boldsymbol{n}_\perpn⊥​ (igen), och för att få fram skjuvspänningen projicerar vi spänningsvektorn på n∥\boldsymbol{n}_\paralleln∥​, vilket är en vektor parallell mot ytan: n∥=(cos⁡30∘sin⁡30∘0) \boldsymbol{n}_\parallel = \begin{pmatrix} \cos 30^\circ\\ \sin 30^\circ\\ 0 \end{pmatrix} n∥​=⎝⎜⎛​cos30∘sin30∘0​⎠⎟⎞​
