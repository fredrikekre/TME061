Räkneövning 3 - Uppgift 8.15 från [1]

Ett slutet tunnväggigt rör belastas av ett inre övertryck ppp och ett vridande moment Mv=πpa3M_v = \pi pa^3Mv​=πpa3, se figur.

a) Bestäm spänningstillståndet; rita Mohrs spänningscirkel; beräkna huvudspänningarna.

b) Beräkna effektivspänning enligt Tresca och enligt von Mise.

Lösning

Denna uppgift är identisk med 8.14 förutom att det också tillkommer en normalspänning längs med röret pga att behållaren är sluten. Ångpanneformlerna ger då

σx=pa2h \sigma_x = \frac{pa}{2h} σx​=2hpa​

och sedan tidigare pga tryck och vridning

σθ=pah, \sigma_\theta = \frac{pa}{h}, σθ​=hpa​, τθx=Mv2πa2h=πpa32πa2h=pa2h. \tau_{\theta x} = \frac{M_v}{2\pi a^2 h} = \frac{\pi pa^3}{2\pi a^2 h} = \frac{pa}{2h}. τθx​=2πa2hMv​​=2πa2hπpa3​=2hpa​.

Övriga spänningar är 0. Vi kan då rita Mohrs spänningscirkel

Huvudspänningar kan räknas ut enligt föregående uppgift genom att ta fram cirkelns mittpunkt:

σavg=σx+σθ2=3pa4h \sigma_{\mathrm{avg}} = \frac{\sigma_x + \sigma_\theta}{2} = \frac{3pa}{4h} σavg​=2σx​+σθ​​=4h3pa​

och cirkelns radie RRR:

R=12(σθ−σx)2+(−τθx−τθx)2=5pa4h R = \frac{1}{2}\sqrt{(\sigma_\theta - \sigma_x)^2 + (-\tau_{\theta x} - \tau_{\theta x})^2} = \frac{\sqrt{5}pa}{4h} R=21​(σθ​−σx​)2+(−τθx​−τθx​)2​=4h5​pa​

Huvudspänningarna fås då som

σ1,2=σavg±R⇒σ1=pa4h(3+5), σ2=pa4h(3−5) \sigma_{1,2} = \sigma_{\mathrm{avg}} \pm R \quad \Rightarrow \quad \sigma_1 = \frac{pa}{4h}(3 + \sqrt{5}),\ \sigma_2 = \frac{pa}{4h}(3 - \sqrt{5}) σ1,2​=σavg​±R⇒σ1​=4hpa​(3+5​), σ2​=4hpa​(3−5​)

Den tredje huvudspänningen för plant spänningstillstånd är 0, så efter att ha numrerat dem i storleksordning har vi

σ1=pa4h(3+5), σ2=pa4h(3−5), σ3=0 \sigma_1 = \frac{pa}{4h}(3 + \sqrt{5}),\ \sigma_2 = \frac{pa}{4h}(3 - \sqrt{5}),\ \sigma_3 = 0 σ1​=4hpa​(3+5​), σ2​=4hpa​(3−5​), σ3​=0

Effektivspänningar fås nu enligt Trescas formulering:

σeTresca=σ1−σ3=pa4h(3+5) \sigma_e^\mathrm{Tresca} = \sigma_1 - \sigma_3 = \frac{pa}{4h}(3 + \sqrt{5}) σeTresca​=σ1​−σ3​=4hpa​(3+5​)

och enligt von Mises formulering:

σevonMise=12(σ1−σ2)2+(σ1−σ3)2+(σ2−σ3)2=6pa2h \sigma_e^\mathrm{vonMise} = \frac{1}{\sqrt{2}} \sqrt{(\sigma_1-\sigma_2)^2 + (\sigma_1-\sigma_3)^2 + (\sigma_2-\sigma_3)^2} = \frac{\sqrt{6}pa}{2h} σevonMise​=2​1​(σ1​−σ2​)2+(σ1​−σ3​)2+(σ2​−σ3​)2​=2h6​pa​
